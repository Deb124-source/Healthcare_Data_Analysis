## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on healthcare datasets to extract meaningful insights that can support:

- Better patient care
- Hospital decision-making
- Cost optimization
- Healthcare trend analysis

The dataset is cleaned, processed, and analyzed using Python data science libraries to uncover patterns in patient demographics, diseases, treatments, and hospital performance.

---

## 🎯 Objectives

- Clean and preprocess raw healthcare data
- Perform Exploratory Data Analysis (EDA)
- Identify disease and patient trends
- Analyze hospital admission & discharge patterns
- Visualize healthcare cost distribution
- Generate actionable insights

---

## 📊 Key Insights

- Age-wise and gender-wise patient distribution  
- Most frequent diseases and diagnoses  
- Hospital admission trends over time  
- Treatment effectiveness patterns  
- Healthcare cost variations and outliers

## 🚀 How to Run This Project

```bash

# Navigate into the project directory
cd Healthcare_Data_Analysis

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter Notebook
jupyter notebook
```
